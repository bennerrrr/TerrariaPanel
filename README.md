# Terraria Server — The Boyz

A self-hosted Terraria server running [TShock](https://github.com/Pryaxis/TShock) in Docker, with a lightweight web panel for server management.

## Features

- Start, stop, and restart the server from the browser
- Live console output (SSE log streaming)
- Online player list with one-click teleport
- Set time of day (dawn / noon / dusk / midnight)
- Save the world on demand
- All config via `.env` — no hardcoded values

## Project structure

## Prerequisites

- [Docker](https://docs.docker.com/get-docker/) and [Docker Compose](https://docs.docker.com/compose/)
- A TShock token (set up during first run — see below)

## Setup

### 1. Clone and configure

```bash
git clone <repo-url>
cd terraria
cp .env.example .env
```

Edit `.env` and set at minimum:

```env
WORLD_FILENAME=your_world.wld
TSHOCK_TOKEN=your_token_here
```

### 2. Get a TShock token

The token is generated by TShock on first start. Run the server once without the web panel to get it:

```bash
docker compose up terraria
```

Watch the logs — TShock will print a setup URL and token. Copy the token into your `.env` file, then stop the container (`Ctrl+C`).

### 3. Start everything

```bash
docker compose up -d
```

The web panel will be available at `http://<your-host>:4823`.

## Configuration

All options live in `.env`:

| Variable | Default | Description |
|---|---|---|
| `WORLD_DIR` | `./world` | Host path to the world directory |
| `WORLD_FILENAME` | `the_boyz.wld` | World file to load |
| `TERRARIA_PORT` | `7777` | Host port for the game server |
| `CONTAINER_NAME` | `terraria` | Docker container name for the game server |
| `TSHOCK_HOST` | `terraria` | Hostname of TShock (internal Docker network) |
| `TSHOCK_PORT` | `7878` | TShock REST API port |
| `TSHOCK_TOKEN` | *(required)* | TShock REST API token |
| `WEB_PORT` | `4823` | Host port for the web panel |

## Usage

### Web panel

Open `http://<host>:4823` in your browser.

| Button | Action |
|---|---|
| ▶ Start | Start the Terraria container |
| ■ Stop | Stop the Terraria container |
| ↺ Restart | Restart the Terraria container |
| 💾 Save | Save the world via TShock |
| Teleport → | Teleport one player to another |
| Time buttons | Set the in-game time |

The console streams live server output. Use the input bar to send any raw TShock command. Click **↓ Scroll to bottom** to jump to the latest output.

### Useful commands

```bash
# View web panel logs
docker logs -f terraria-web

# View game server logs
docker logs -f terraria

# Rebuild and redeploy the web panel after changes
docker compose build terraria-web && docker compose up -d terraria-web

# Stop everything
docker compose down
```
